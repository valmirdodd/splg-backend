# seplag-backend
API que serve os dados ao front-end do teste da seplag

Esta API foi criada como parte do teste para a Seplag e objetiva servir ao front-end do mesmo teste uma forma simples de acesso aos dados da aplicação.
Ela foi desenvolvida em noje.js e possui as seguintes urls

http://localhost:3001/servidores

http://localhost:3001/categorias_beneficios

http://localhost:3001/locais_tramitacao

http://localhost:3001/beneficios

http://localhost:3001/documentos_beneficios

http://localhost:3001/tramitacao

para iniciar, utilizar o seguinte comando: npm start
